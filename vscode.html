Visual Studio Code

빨강 처음 프로젝트를 시작할 때, 원하는 곳에 폴더를 만든 후, 파일->폴더열기를 클릭하고 폴더를 선택하면 노랑부분에 폴더가 열린다. 실행 버튼을 눌러 프로그램을 실행시키거나 터미널 버튼으로 파랑 부분에 새로운 터미널을 열 수도 있다
주황 1번째 아이콘 : 프로젝트 폴더를 확인할 수 있다
2번째 아이콘 : 검색할 수 있다
3번째 아이콘 : Git과 연결되어 폴더에서 변경점이 있는 부분을 표시한다.
4번째 아이콘 : 디버그 정보를 표시한다.
5번째 아이콘 : 다양한 기능들을 설치할 수 있다.
노랑 폴더 또는 파일을 편집할 수 있다.
초록 해당 부분에서 코드를 편집한다.
파랑 터미널 창이다. 파이썬 등의 언어 실행 시, 해당 창에서 프로그램이 실행된다.
보라 오류가 발생했을 시 표시된다.
