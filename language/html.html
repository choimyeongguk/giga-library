HTML(Hyperltext Markup Language)

엄연히 말하자면 HTML은 프로그래밍 언어가 아닌 사용자가 보는 페이지가 어떻게 만들어졌는데 브라우저가 확인하게 하는 마크업 언어이다.
HTML은 웹페이지의 뼈대를 만드는데 사용된다.
HTML을 이용해 대략적인 페이지를 만들고 CSS를 통해 사용자가 보기 편하도록 꾸민다.

HTML은 기본적으로 '태그'로 이루어진다. 태그의 종류와 안에 들어가는 내용에 따라 다양한 요소를 만들 수 있다.

<태그명>내용</태그명>

HTML 문서의 기본 구조

<!DOCTYPE html>
<html>
  <head>
   ...
  </head>
  <body>
   ...
  </body>
</html>


HTML 문서의 기본 구조는 위와 같다.

태그설명
<!DOCTYPE html>현재 문서가 HTML5 언어로 작성한 웹 문서임을 나타낸다. HTML 문서를 작성할 때 기본적으로 적어주어야 한다.
<html> ... </html>웹 문서의 시작과 끝을 나타내는 태그이다. head태그와 body태그를 감싸며, 웹 브라우저가 <html> 태그를 만나면 </html> 까지 소스를 읽어 화면에 표시한다.
<head> ... </head>웹 브라우저가 웹 문서를 해석하는 데 필요한 정보를 입력한다.
<body> ... </body>실제로 웹 브라우저 화면에 나타나는 내용이다.
